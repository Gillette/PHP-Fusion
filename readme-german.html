1. Einleitung
PHP-Fusion ist ein simples open-source content management system (CMS), geschrieben in PHP. Es benutzt eine mySQL Datenbank um den Inhalt einer Homepage zu speichern und beinhaltet ein einfaches aber umfangreiches Administrations-System. PHP-Fusion enthält alle wichtigen Funktionen die auch in anderen CMS zu finden sind.

2. Installation
  1. Lade den Inhalt des "php-files"-Ordner auf deinen Server.
  2. Benenne blank_config.php zu config.php um.
  3. CHMOD die folgenden Dateien und Ordner auf 777:
    • administration/db_backups/
    • images/
    • images/imagelist.js
    • images/articles/
    • images/avatars/
    • images/news/
    • images/news_cats/
    • images/photoalbum/
    • images/photoalbum/submissions/
    • forum/attachments/
    • config.php
  4. Beende den Setupprozess indem du den Anweisungen folgst.
  5. CHMOD config.php wieder zu 644 und lösche setup.php von deinem Server.
3. Upgrade von v6.00.1xx, v6.00.2xx oder v6.00.3xx
Bevor sie ein Update auf die neue Version tätigen, empfehlen wir ihnen ein vollständiges Backup ihrer Seite anzulegen (Dateien und Datenbank). Während des Upgrade-Prozesses werden die Fotogallerydaten in der Datenbank umbenannt. Du kannst deine alten Fotos und Fotoalben mithilfe der "Photo Converter"-Infusion, erhältlich auf www.php-fusion.co.uk, in die neue Gallery importieren.
Sie sollten zur Kenntnis nehmen das dieses Release einige strukturelle Änderungen einführt und einige Funktionen nicht einwandfrei ihren Dienst erfüllen könnten solange nicht alle Teile von PHP-Fusion auf den neusten Stand gebracht wurden.
  1. Lade upgrade.php aus dem "upgrade"-Ordner in den "administration"-Ordner auf deinen Server.
  2. Logge dich auf deiner Seite als Super Administrator ein und drücke "Upgrade" im Adminbereich.
  3. Folge den Anweisungen bis "Database upgrade complete" erscheint.
  4. Lade den Inhalt des "php-files"-Ordner auf deinen Server.
  5. Vergewissere dich das folgende Ordner und Dateien auf CHMOD 777 stehen:
    • images/photoalbum/submissions/
    • images/news_cats/ (v6.00.1x only)
    • images/imagelist.js (v6.00.1x only)
4. Security tips
Hier sind einige nützliche Tips um deine Seite sicher zu halten:
  • Vergewissere dich das die config.php schreibgeschützt ist (sollte CHMODed auf 644 sein).
  • Lasse niemals die setup.php auf dem Server wenn PHP-Fusion fertig installiert ist.
  • Stelle sicher das FTP und MySQL Passwörter verschieden sind.
  • Lasse niemals Forum-Anhänge folgender Typen zu: php, html, exe, oder irgendeines Textformates.
5. Support sites
Sollten Fragen oder Probleme bestehen kannst du die Hauptseite von PHP-Fusion unter www.php-fusion.co.uk aufsuchen und eine Nachricht im Forum hinterlassen. Außerdem kannst du einen IRC Chat unter irc.outerweb.org/phpfusion aufsuchen.

PHP-Fusion kann durch sogenannte Infusions erweitert werden. Dies sind Plugins welche sich sehr einfach installieren lassen. Eine große Anzahl Infusions gibt es auf www.phpfusion-mods.com.

Wenn dir die mitgelieferten Themes nicht gefallen kannst du unter www.phpfusion-themes.com neue Themes suchen und downloaden.

PHP-Fusion hat außerdem Supportseiten in verschiedenen Sprachen z.B. Arabia, Belgium, Bulgaria, Bosnia, Croatia, Denmark, France, Germany, Hungary, Iran, Italy, Lithuania, Netherlands, Poland, Russia, Sweden, Taiwan, and Turkey
