1. Εισαγωγή
Το PHP-Fusion είναι ένα όλα-σε-ένα open-source content management σύστημα (CMS) που γράφτηκε με PHP. Χρησιμοποιεί μια βάση δεδομένων mySQL για να αποθηκεύσει τα δεδομένα της σελίδας και περιλαμβάνει ένα απλό περιεκτικό σύστημα adminstration. Το PHP-Fusion περιλαμβάνει τα πιο κοινά χαρακτηριστικά που θα αναμένατε να δείτε σε πολλά άλλα συστήματα CMS.

To PHP-Fusion εκδόθηκε υπό τους όρους της Έκδοσης 2 GNU General Public License. Για περισσότερες πληροφορίες παρακαλώ επισκεφτείτε www.gnu.org ή πηγαίνετε στο αρχείο gpl.txt το οποίο υπάρχει στον Φάκελο του PHP-Fusion. Έχετε το δικαίωμα να αλλάξετε τον Κώδικα όμως δεν σας επιτρέπεται να το μεταδώσετε ως δικό σας προϊών. Θα εκτιμούσαμε εάν διατηρούσατε το "Powered by PHP-Fusion" στο Τέλος της σελίδας. Αισθανόμαστε ότι αυτό είναι δίκαιο διότι αν σκεφτείτε το πόσες εκατοντάδες ώρες εργασίας χρειάστηκαν για την ανάπτυξη αυτού του προγράμματος. Εντούτοις, εάν αφαιρέσετε την σημείωση πνευματικών δικαιωμάτων θα σας παρακαλούσα να στείλετε μια Δωρεά μέσω Paypal.
2. Εγκατάσταση
  1. Ανεβάστε το περιεχόμενο του Φακέλου php-files στον κεντρικό Υπολογιστή σας.
  2. Μετονομάστε το blank_config.php σε config.php.
  3. Δώστε δικαιώματα 777 στους παρακάτω φακέλους και αρχεία:
    • administration/db_backups/
    • images/
    • images/imagelist.js
    • images/articles/
    • images/avatars/
    • images/news/
    • images/news_cats/
    • images/photoalbum/
    • images/photoalbum/submissions/
    • forum/attachments/
    • config.php
  4. Πηγαίνετε στην σελίδα σας και τρέξτε το setup.php π.χ. http://www.ονομα.gr/setup.php.
  5. Ολοκληρώστε την Εγκατάσταση Βήμα - Βήμα όπως σας ζητείται.
  6. Δώστε δικαίωμα 644 στο config.php ΚΑΙ διαγράψτε το setup.php από τον κεντρικό Υπολογιστή σας.
3. Αναβάθμιση από v6.00.1xx, v6.00.2xx ή v6.00.3xx
Πριν ξεκινήσετε την αναβάθμιση προτείνουμε να σώσετε τα αρχεία και την Βάση Δεδομένων σας. Κατα την διάρκεια της αναβάθμισης, οι πίνακες της Βάσης Δεδομένων που βρίσκεται το περιεχόμενο του ’λμπουμ Φωτογραφιών σας θα μετονομαστεί. Μπορείτε να εισάγετε τα παλιά Φώτο-’λμπουμ σας και Φωτογραφίες αν χρησιμοποιήσετε το Εργαλείο με όνομα photo converter και είναι διαθέσιμο στο
www.php-fusion.co.uk. Παρακαλώ σημειώστε ότι αυτή η Έκδοση διάφορες δομικές αλλαγές και ως αποτέλεσμα μερικά στοιχεία στην Σελίδα σας να μην λειτουργήσουν σωστά μέχρι που να ενημερώσετε όλα τα αρχεία σας ολοκληρωτικά.
  1. Ανεβάστε το upgrade.php από τον Φάκελο upgrade παράλληλα στην τρέχουσα Έκδοση στον Φάκελο administration.
  2. Συνδεθείτε στην σελίδα σας ως Κύριος Διαχειριστής και πατήστε Αναβάθμιση ή Upgrade μέσω της Επιτροπής Διαχειριστών.
  3. Ακολουθείστε όλες τις κινήσεις που σας ζητείται μέχρι να ολοκληρωθεί η Αναβάθμιση".
  4. Ανεβάστε όλο το περιεχόμενο του Φάκελου php-files στον κεντρικό Υπολογιστή σας.
  5. Βεβαιωθείτε ότι οι ακόλουθοι Φάκελοι και αρχεία έχουν δικαίωμα 777:
    • images/photoalbum/submissions/
    • images/news_cats/ (v6.00.1x μόνο)
    • images/imagelist.js (v6.00.1x μόνο)
4. Συμβουλές ασφαλείας
Παρακάτω θα δείτε μερικές χρήσιμες συμβουλές για να διατηρήσετε την Σελίδα σας ασφαλές:
  • Βεβαιωθείτε ότι το config.php δεν έχει δικαιώματα πρόσβασης για το κοινό (δώστε 644 για δικαίωμα πρόσβασης).
  • Μην αφήνετε ποτέ το setup.php στον κεντρικό Υπολογιστή σας όταν το PHP-Fusion είναι εγκατεστημένο.
  • Σιγουρευτείτε ότι οι κωδικοί του FTP  και του MySQL σας είναι διαφορετικοί.
  • Μην επιτρέπεται ποτέ συνάψεις όπως php, html, exe, ή κάτι παρόμοιο που έχει σχέση με αρχείο κειμένου.
5. Σελίδες υποστήριξης
Αν έχετε ερωτήσεις ή προβλήματα περί του PHP-Fusion, παρακαλώ επισκεφτείτε την κεντρική σελίδα στο www.php-fusion.co.uk ( για ελληνική υποστήριξη στο www.php-fusion.gr ) και στείλτε ένα μήνυμα στο Φόρουμ. Διαφορετικά μπορείτε να μπείτε και στο κανάλι υποστήριξης του IRC Chat στο irc.outerweb.org/phpfusion.

Το PHP-Fusion μπορεί να επεκταθεί με πρόσθετες τις λεγόμενες Επεκτάσεις, τα οποία είναι Plugins που μπορείτε πολύ εύκολα να τα εγκαταστήσετε. Μπορείτε να βρείτε μια μεγάλη συλλογή Επεκτάσεων στην σελίδα www.phpfusion-mods.com και για ελληνικά στο www.php-fusion.gr.

Αν δεν είστε ικανοποιημένοι με τα που υπάρχουν ίδει στο PHP-Fusion τότε επιστεφτείτε την σελίδα  www.phpfusion-themes.com όπου θα βρείτε μια μεγάλη συλλογή από Themes υψηλής ποιότητας.

Το PHP-Fusion έχει έναν ικανοποιητικό αριθμό Ιστοσελίδων υποστήριξης για πολλές Γλώσσες με Βάση την Αραβία, Βέλγιο, Βουλγαρία, Βοσνία, Κροατία, Δανία, Γαλλία, Γερμανία, Ελλάδα, Ουγγαρία, Ιράν, Ιταλία, Λιθουανία, Ολλανδία, Πολωνία, Ρωσία, Σουηδία, Ταιβάν, και Τουρκία
