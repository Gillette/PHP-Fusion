ناساندن
PHP-Fusion سیستەمێکی سوکەڵە و سەرچاوەکراوەی بەڕێوەبردنی ناوەڕۆکی ماڵپەڕە (CMS). بە PHP5 نووسراوە (گونجاو لەگەڵ PHP4). پێویستی بە بنکەدراوەیەکی MySQL ـە بۆ پاشەکەوتکردنی ناوەڕۆکی ماڵپەڕەکەت و سیستەمێکی بەڕێوەبردنی بەهێز و سادەی هەیە. PHP-Fusion زۆربەی ئەو تایبەتمەندییە باوانەی تیادایە کە چاوەڕوان دەکەیت لە هەر گورزەیەکی CMS ـی تردا هەبێت.

دامەزراندن
پێش دامەزراندنی PHP-Fusion پێویستە بنکەدراوەیەکی MySQL دروستبکەیت. ئەتوانیت ئەم کارە بکەیت لە ڕێی پانێڵی دەستبەسەرداگرتنی خانەخوێی ماڵپەڕەکەتەوە یان phpMyAdmin. دڵنیابە لەوەی زانیارییەکانی چوونەژوورەوەی mysql ـت هەیە لە ناویاندا ناوی خانەخوێ، ناوی بەکارهێنەر، تێپەڕەوشە و ناوی بنکەدراوە پێویستە ئەمانە دیاری بکەیت لە کاتی دامەزراندن.

1. پێش ئەوەی پەڕگەکان بار بکەیت، پەڕگەی _config.php (کەوتۆتە ناو بوخچەیەکەوە بە ناوی files) بۆ config.php.

2. ناوەڕۆکی بوخچەی files بار بکە بۆ ڕاژەکاری وێبگەکەت.

3. پێش ئەوەی PHP-Fusion کارپێبکەیت لەسەر ڕاژەکارێکی نێوخۆیی، لە زۆربەی بارەکاندا پێویستە CHMOD ـی ئەم پەڕگە و بوخچانە بگۆڕیت بۆ 777:

  • administration/db_backups/
  • images/
  • images/imagelist.js
  • images/articles/
  • images/avatars/
  • images/news/
  • images/news_cats/
  • images/photoalbum/
  • images/photoalbum/submissions/
  • forum/attachments/
  • config.php

4. بڕۆ بۆ ماڵپەڕەکەت کە setup.php پێویستە خۆکارانە بکرێتەوە. ئەگەر نا، خۆت دەستکارانە setup.php بکەیتەوە بە لێدانی url ـی تەواوەتی ماڵپەڕەکەت بەدوایدا /setup.php. نمونە: http://www.mallperreket.com/setup.php

5. پرۆسەی دامەزراندن تەواو بکە.

6. یەکسەر پاش کۆتاییهاتنی دامەزراندنی PHP-Fusion پێویستە CHMOD ـی config.php بگێڕیتەوە بۆ 644 و setup.php بسڕیتەوە ڕاژەکاری وێبگەکەت.

نوێکردنەوە لە وەشانی V6.01.12+ ـەوە
پێش ئەوەی نوێی بکەیتەوە بە توندی ئاگادارت ئەکەینەوە کە پشتگیری پەڕگەکان و بنکەدراوە بکەیت چونکە PHP-Fusion 7 نوێکردنەوەیەکی گەورەیە بە بەراورد لەگەڵ وەشانەکای پێشتر. پێوستە بە وردی ئەم ڕێبەرییە جێبەجێ بکەیت.

1. کۆدەکانی وەشانی 6 ڕاستەوخۆ لەگەڵ وەشانی 7 ناگونجێت. پێوسیتە پێوەکراو، پانێڵەکان یان ڕووکارەکان نوێبکەیتەوە پێش نوێکردنەوەی وەشان یان وەشانێکی تەواوی 7 بەدەست بهێنیت. ئامۆژگاریکراوە کە هەرچۆنێک بێت، هەموو کۆدەکانی بەشی سێیەم ناچالاک بکەیت بۆ نەهێشتنی روودانی هەر جۆرە کێشەیەک.

2. یەکەم پێویستە سکریپتی نوێکردنەوەی وەشان بار بکەیت لەو بوخچەیەی بە ناوی 'upgrade v601x' ـە بۆ بوخچەی بەڕێوەبەرایەتی لە وێبگەکەت.

3. بچۆ ژوورەوە بۆ ماڵپەڕەکەت وەک بەڕێوەبەری گشتی. لە ژێر تابی بەڕێوەبەرایەتی سیستەم لە پانێڵی بەڕێوەبەرایەتی کرتە لە نوێکردنەوەی وەشان بکە پاشان کرتە لە ئەو دوگمەیە بکە بە ناوی نوێکردنەوە. پێویستە نوێکردنەوەکە جێبەجە بکەیت لە چەند هەنگاوێکدا پڕۆسەی نوێکردنەوەی وەشان تەواو دەبێت تەنها کاتێک کە دەستەواژەی 'نوێکردنەوەی بنکەدراوە تەواوبوو' دەبینیت.

4. زۆر گرنگ: ئەگەر وەشانەکە ژمارەیەک لە گۆڕانکاریی پێکهاتەیی تیادایە، هەندێک لە توخمەکانی ماڵپەڕەکەت بە باشی کار ناکەن هەتا هەموو پەڕگەکانت نوێ ئەکەیتەوە. پێویستە هەموو پەڕگەکانی بوخچەی files باربکەیت.

گەر تا ئێستا هەر بە گومانیت تکایە ئازادبە لە هەر پرسیارێک لە ماڵپەڕە پاڵپشتیکردنەکانمان، دەیان ئەندامی بەتوانای هەیە لە کۆمەڵەکەمان کە ئەتوانن هاوکاری و ئامۆژگاریت بکەن لە پڕۆسەی نوێکردنەوەی وەشاندا.

سەرنجەکانی ئاساییش
ئەمەش چەند سەرنجێکی بەسوودن تا بتوانیت ماڵپەڕەکەت بە سەلامەتی بپارێزیت:
  • دڵنیابە لەوەی پەڕگەی config.php توانای نووسینەوەی لەسەر نییە (پێویستە CHMOD ـی بگۆڕدرێت بۆ 644).
  • بیرت نەچێت کە پەڕگەی setup.php بسڕیتەوە هەر کاتێک PHP-Fusion ـت دامەزراند.
  • هەمیشە دڵنیابە لەوەی تێپەڕەوشەکانی FTP و MySQL جیاوازن.
  • هەرگیز مەهێڵە پەڕگەی php، html، exe، یان هەر جۆرە پەڕگەیەکی دەقی لە مەکۆ هاوپێچ بکرێت.
وێبگە پاڵپشتیکەرەکان
گەر هەر پرسیار و کێشەیەکی پەیوەندیدارت بە PHP-Fusion ـەوە هەیە، تکایە سەردانی ماڵپەڕی سەرەکی پەرەپێدان بکە لە www.php-fusion.co.uk و پەیامێک بنێرە لە مەکۆکانی پاڵپشتیکردن. ئێمە تیمێکی یاریدەدەری تایبەتمان هەیە کە ئامانجی ئەوەیە هەر کێشەیەکت هەیە چارەسەری بکەت لە 48 کاژێردا.

دەتوانرێت PHP-Fusion بەرفراوان بکرێت بە زیادکردنی پێوەکراو، ئەم پێوەکراوانە زۆر ئاسانن بۆ دامەزراندن. ئەتوانیت پێوەکراوە جۆربەجۆرەکانت لە www.phpfusion-mods.com دەست بکەوێت.

گەر حەزت بە ڕووکاری سەرەکی PHP-Fusion نییە ئەتوانیت سەردانی ماڵپەڕی فەرمی ڕووکارەکانی PHP-Fusion بکەیت لە themes.php-fusion.co.uk کە ژمارەیەکی زۆر ڕووکاری قەشەنگت دەست دەکەوێت.

هەروەها PHP-Fusion ژمارەیەک ماڵپەڕی پاڵپشتیکەری فەرمی هەیە بە زمانە جیاوازەکانی ئەم وڵاتانە: عەرەب، بەلجیکا، بەڕازیل، بولگاریا، دانیماڕک، کۆماری چیک، فەڕەنسا، ئەڵمانیا، هەنگاریا، ئێران، ئیتاڵیا، کوردستان، هۆڵەندا، نەرویژ، پۆلەندا، ڕۆمانیا، ڕووسیا، سلۆڤاکیا، ئیسپانیا، سوید و تورکیا.
