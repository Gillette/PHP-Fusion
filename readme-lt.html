INSTALIAVIMAS
Prieð instaliuojami PHP-Fusion, pirmiausia turësite susikurti MySQL duomenø bazæ. Tai padaryti galite pasinaudojæ savo hostingo administracijos panele, arba PHPMyAdmin duom. baziø valdymo árankiu. Prisiminkite savo MySQL duomenø bazës duomenis (prisijungimo vardà, slaptaþodá, duom. bazës pavadinimà bei duom. bazës serverio adresà), kadangi jie bus bûtini vykdant instaliacijà.

1. Prieð keldami failus á FTP serverá, pervadinkite failà _config.php (aplankas: php-files) á config.php.

2. Ákelkite failus á savo FTP serverá.

3. Suteikite CHMOD 777 atþymas ðiems failams:

  • administration/db_backups/
  • images/
  • images/imagelist.js
  • images/articles/
  • images/avatars/
  • images/news/
  • images/news_cats/
  • images/photoalbum/
  • images/photoalbum/submissions/
  • forum/attachments/
  • config.php

4. Eikite á savo puslapá, setup.php failas turëtø atsiverti automatiðkai. Jeigu taip neávyks, turite suvesti adresà iki setup.php failo, pvz.: http://www.Jûsø-Tinklapis.lt/setup.php.

5. Suveskite MySQL duomenø bazës prisijungimo duomenis.

6. Instaliavæ Php-Fusion turinio valdymo sistemà, suteikite CHMOD 644 atþymas failui config.php, bei iðtrinkite setup.php failà ið serverio.

ATNAUJINIMAS IÐ V6.01.12+
Prieð atnaujinant, rekomenduojame pasidaryti savo duomenø bazës kopijà. Tuomet ádëmiai sekite ðias instrukcijas.

1. V6 kodas nesuderinamas su v7 kodu. Privalote atnaujinti visus modulius(MODS), papildinius(INFUSIONS) bei dizaino ðablonus(THEMES). Rekomenduojame paðalinti visà treèiøjø ðaliø kodà ið Jûsø tinklapio.

2. Pirmiausia ákelkite á FTP failus ið aplanko 'upgrade v601x', á administracijos aplankà serveryje.

3. Prisijunkite savo svetainëje kaip Super Administratorius. Administracijos panelëje renkamës "Sistemos administracija", spaudþiame "Atnaujinimas", tada spaudþiame ant mygtuko "Atnaujinti". Atidþiai sekite visus atnaujinimo þingsnius,procesas bus pilnai atliktas tokiu atveju, kai jûs pamatysite "Duomenø bazë atnaujinta", ar kaþkà panaðaus.

4. SVARBU: Ðio leidimo struktûra kitokia, todël turësite atnaujinti visus failus, antraip sistema gali veikti nesklandþiai. PRIVALOTE ákelti VISUS failus ið failø aplanko.

5. Jei viskà atlikote, suteikite CHMOD 777 atþymas Jûsø FTP direktorijai images/photoalbums/submissions.

Jeigu nepavyko, praðome klausti patarimo viename ið mûsø pagalbos tinklapiø, administratoriai, bei kiti vartotojai mielai jums padës atnaujinti versijà.
